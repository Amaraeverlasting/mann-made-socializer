Connecting a client account (The Exponentials):

1. Open Chrome and create a new Profile for the client
2. Log into their social accounts in that profile
3. Start Chrome with remote debugging:
chrome --remote-debugging-port=18801 --user-data-dir=/tmp/client-chrome

4. Save the session:
agent-browser --cdp 18801 state save ~/.openclaw/workspace/browser-sessions/exponentials-x-session.json

5. Mark the account as active in the Accounts tab
